Project Overview

A mindfulness and meditation app designed for young adults struggling with stress, motivation, and sleep. As my Advanced Diploma capstone project, I handled full-stack development including UX design, user testing, and implementation.

Built as a functioning application using Xcode with custom content and interactions, Quiet features intelligent gesture recognition and personalized meditation experiences tailored to individual user needs and preferences.

Technical Innovation

Implemented intelligent gesture recognition for hands-free meditation control, allowing users to navigate the app without breaking their mindfulness state. This required balancing technical complexity with intuitive user experience.

The 12-week development cycle allowed for extensive user research, iterative design, and comprehensive testing. This project demonstrated my ability to manage the complete product development lifecycle from concept to functional application.
