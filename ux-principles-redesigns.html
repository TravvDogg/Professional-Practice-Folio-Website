"Tandem redesign of RMIT Enrolment & Amazon Homepage"

Duration

12 weeks

Role

UX Designer & Researcher

Tools

Figma, User Research, A/B Testing

Platform

Web Applications

Focus

UX Principles Application

Project Overview

Redesigned both RMIT's Enrolment Online website and Amazon's homepage using UX Design principles to improve user interaction flow, increase efficiency, and reduce error rates. Heavy focus on external user research (quantitative and qualitative), iterative development with concurrent user research, and A/B testing to quantify improvements.

This dual redesign project allowed me to explore how fundamental UX principles apply differently across various platforms and user contexts - from academic enrollment systems to e-commerce platforms.

Research-Driven Approach

Conducted extensive user research including quantitative analysis, qualitative interviews, and A/B testing to validate design decisions. The project emphasized evidence-based design improvements over aesthetic changes.

The 12-week timeline enabled comprehensive research phases, iterative design development, and thorough testing of proposed solutions. This project demonstrated how systematic application of UX principles can significantly improve user experiences across different digital environments.
